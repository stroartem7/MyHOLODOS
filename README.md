# MyHOLODOS 🧊

Android application for recipe management and AI-powered meal recommendations.

## Features

* Browse dishes
* AI recommendations via YandexGPT
* Shopping list
* Favorite recipes
* Local Room database
* Modern UI with Jetpack Compose

## Tech Stack

* Kotlin
* Jetpack Compose
* Room
* Retrofit
* Coroutines
* Navigation Compose
* YandexGPT API

## Architecture

* UI (Compose)
* Repository Layer
* Room Database
* Retrofit API Layer
